**** grep/egrep 中如何匹配tab                                       :grep:
    - input tab with ^v^i(control-v control-i)
    - grep $'\t' file
**** grep常用的选项                                                 :grep:
    - C NUM 显示出匹配行附近的 NUM 行上下文。
    - n 显示行号。
    - H 显示文件名。这两个选项对于在一大堆文件里面搜索东 西非常有用。
    - o 只显示匹配的部分，这对于从一大堆东西中提取某些特殊信息非常有用。
    - i 忽略大小写
      - v 反向匹配，即显示不匹配的行。
