*** header argument
**** :exports
  1. :exports code
  2. :exports results
  3. :exports both
  4. :exports none
**** :results
there are 3 kinds of :results header argument:
1. collections
2. type
3. handling
***** collections
1. value
2. output
***** type
1. table
2. list
3. verbatim
4. raw
5. latex
6. html
7. code
***** handling
1. silent
2. replace
3. append
4. prepend

***** for example
:results output replace raw
